USER GUIDE
How to use the Highway Quality Inspection System
🛣️ Dashboard
Filter by Year, Month, Dzongkhag, and RO. All KPI cards and breakdown tables auto-update based on your selection.
➕ New Entry + Auto QC
Select Work Type and enter test results. The system automatically checks against SBRW 2025 / Chapter 21 thresholds and shows PASS / FAIL instantly. Manual override available.
🔬 SBRW QC Tool
Full SBRW 2025 Chapter 21 test tool with all 209 Road Works items. Enter measured values and get instant Pass/Fail against standard thresholds. Generate full QC reports.
⚠️ ATR Tracker
All defect entries are auto-shown here. 🔴 OVERDUE | 🟠 DUE SOON (≤7 days) | 🟡 Pending | 🟢 Submitted. Click Mark Submitted to update status.
📈 Analytics
4 charts: Monthly trend, Pass vs Fail doughnut, Dzongkhag bar, RO comparison. Auto-update with each new entry.
🔐 Login & Roles
Admin — full access. User (Dzongkhag/RO/Thromde) — add project updates, print reports, location locked. Viewer — read-only access via Google.
